# **My Azure Developer Associate Journey**
## **Overview**

This repository highlights my comprehensive learning and hands-on experience in Java Backend Development and Microsoft Azure Cloud Development.

The goal of this journey is to master both backend engineering and cloud technologies, enabling the development of scalable, secure, and production-ready applications.
